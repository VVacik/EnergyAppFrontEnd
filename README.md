# Energy App Frontend

Frontend for visualizing the energy mix and optimal energy usage window.  
Built with **React + Vite** and deployed on **Render.com**.

---

## Technologies

- **React 18**
- **TypeScript**
- **Vite**
- **Fetch API**
- **Render.com**

---

## Local Setup

1. Navigate to the frontend folder:
   cd frontend
   npm install
   npm run dev
Default URL: http://localhost:5173

##API Configuration
The backend URL is configured via environment variables.
VITE_API_URL=http://localhost:5108/energy


## Project Structure
src/
 ├── components/   # UI components
 ├── api/          # Backend communication
 ├── pages/        # Application views/pages
 ├── App.tsx       # Root component
 ├── main.tsx      # Entry point
